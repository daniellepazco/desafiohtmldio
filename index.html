O que são Atributos?

INSTRUÇÕES:

Atributos são comandos que colocamos nas tags para que ela retorne com o resultado esperado. Existem atributos que funcionam em todas as tags, e outros que são específicos para cada tag. Exemplo class, id, lang, style ou específicos como src, disabled, href, label etc...

O que são Elementos?

INSTRUÇÕES:

Os elementos são os comandos em que construímos a nossa página. Exemplo h1, h2, h3 etc...

O que é uma Tag?

INSTRUÇÕES:

Utilizamos as tags para informar ao navegador a estrutura do nosso site. O navegador interpreta produzindo a estrutura e o conteúdo visual da página. Exemplo html, body, head etc...
